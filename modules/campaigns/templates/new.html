Use the form below to automatically create a project for marketing campaign. The list of clients you specify will be used to create a separate ticket for every client.

Alternatively, you can provide a list of contracts, so tickets will be linked to contracts instead. You can even combine the lists, but in this case you are fully responsible for ticket targets.

You will be marked as creator of the project, so you will always have possibility to see project progress in Tickets :: My Projects section.

Title This will become a title for project, automatically created for you.
Clients Paste list of clients (e.g. from Excel), one client per row
Contracts Paste list of contract (e.g. from Excel), one contract per row
Responsible Moderator of new project, who can redefine handlers for tickets (by default, he is the handler)
Due Date DueDate for project's tickets to be closed
Summary Standard Summary for each ticket to be created
